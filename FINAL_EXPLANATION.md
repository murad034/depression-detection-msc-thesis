## Dataset

- **File**: `Student_Depression_Dataset.csv`
- **Instances**: 539 students
- **Features**: 23 attributes (26 after encoding)
- **Class Distribution**: 40.63% Depression (Yes), 59.37% No Depression
- **Generated by**: `generate_clean_dataset.py` with thesis-matching distributions

---

## Preprocessing Pipeline

1. **Binary Encoding** - Yes/No columns to 1/0 (13 columns)
2. **One-Hot Encoding** - Gender, Personality Type, Comfortable Environment
3. **Median Imputation** - Fills missing numeric values
4. **StandardScaler** - Normalizes all features to zero mean, unit variance
5. **Result**: 26 features used for model training

---

## Model Configuration

| Model               | Key Parameters                                     |
| ------------------- | -------------------------------------------------- |
| SVM                 | kernel=rbf, C=10, gamma=scale                      |
| Logistic Regression | max_iter=1000, C=1.0                               |
| XGBoost             | n_estimators=200, max_depth=6, learning_rate=0.1   |
| MLP                 | hidden_layer_sizes=(128, 64, 32), max_iter=500     |

---

## Validation Strategy

- **10-Fold Stratified Cross-Validation** (as specified in thesis)
- **80/20 Train-Test Split** for final evaluation
- Stratification ensures balanced class distribution in every fold
